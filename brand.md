# Brand — Divindex

_Status: active_

**Divindex** = *Divine* (the premium Path of Exile 2 currency) + *index*. It reads
like a market index — finance-credible, not gambling-flavored — which is exactly
the positioning we want with players and with Grinding Gear Games.

**One-liner:** The Path of Exile 2 economy, as an index.
**Tagline:** Track every orb. Forecast every move.

## What it is
A real-time currency index, deep price history, and a free weekly forecasting
game for the Path of Exile 2 economy — built on volume-weighted data from the
game's official in-game Currency Exchange.

## Voice
Concise, confident, trader-literate. Active voice. We talk about *markets,
rates, indices, and forecasts* — never *bets, odds, or wagers*. The Forecast is
a free game of skill: no entry fee, no payouts, ranks and bragging rights only.

## Type
- **Display / wordmark:** Fraunces (serif) — old-world, scholarly, suits the setting.
- **UI / body:** Inter.
- **Numbers / prices / data:** JetBrains Mono.

## Logo
A single golden orb (radial gradient) beside the wordmark. See `assets/favicon.svg`.

## Hard rules (legal / GGG)
- Every page carries: *"Divindex is an independent project and is not affiliated
  with or endorsed by Grinding Gear Games in any way."*
- Use "Path of Exile 2" descriptively only. Do **not** use GGG's coined proper
  nouns (Wraeclast, Kalandra, Vaal, Oriath, etc.) in committed brand copy.
- The Forecast is always presented as free, skill-based, no payouts, no wagering.
